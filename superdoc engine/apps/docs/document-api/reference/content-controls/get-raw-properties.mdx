## Summary

Get the raw sdtPr properties of a content control as a passthrough hash.

- Operation ID: `contentControls.getRawProperties`
- API member path: `editor.doc.contentControls.getRawProperties(...)`
- Mutates document: `no`
- Idempotency: `idempotent`
- Supports tracked mode: `no`
- Supports dry run: `no`
- Deterministic target resolution: `yes`

## Expected result

Returns a ContentControlsGetRawPropertiesResult with the raw properties.

| Field | Type | Required | Description |
