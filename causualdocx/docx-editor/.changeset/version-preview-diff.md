Version history: clicking a saved version now opens a full-canvas, read-only preview of that version with the changes since the previous version overlaid inline (insertions underlined per author, deletions struck through), plus a "Show changes" toggle and a "Restore this version" action. Replaces the cramped monospace diff box in the side rail. The Versions rail is also restructured to match Google Docs: a pinned "Current version" row, per-row kebab (⋮) menu consolidating Restore / Rename / Delete, and an "Only named versions" filter.
